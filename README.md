# kyc-sample

### How To Run
* Change payload inside bio.js (for biometrik and liveness.js for liveness)
* Prepare image file (default tes.jpeg)
```
npm install

#Biometrik test
node bio.js
#Liveness test
node liveness.js
```

### Environment
Don't forget to add .env
```
API_KEY=
SECRET_KEY=
TOKEN=
```
